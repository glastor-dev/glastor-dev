# Soporte

Si necesitas ayuda con **GLASTOR README Generator**, aquí están los canales recomendados:

## 1) Preguntas y ayuda general
- Abre un **Issue** en GitHub con la etiqueta `question` (si existe) o descríbelo como pregunta.
- Incluye: versión de Python, sistema operativo, comando ejecutado y el error completo.

## 2) Reporte de bugs
- Crea un **Issue** con:
  - Pasos para reproducir
  - Comportamiento esperado vs. actual
  - Logs / stacktrace
  - Archivos de ejemplo (si aplica)

## 3) Propuestas y mejoras
- Abre un **Issue** describiendo el caso de uso y el beneficio.
- Si ya tienes una idea de implementación, añade un bosquejo.
